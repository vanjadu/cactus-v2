## Environment Variables

To run this project, you will need to add the following Firebase environment variables to your .env file

`API_KEY`

`AUTH_DOMAIN`

`AUTH_DOMAIN`

`ROJECT_ID`

`STORAGE_BUCKET`

`MESSAGING_SENDER_ID`

`APP_ID`

`MEASUREMENT_ID`
